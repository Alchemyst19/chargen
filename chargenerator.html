About

chargen is Harrison's pet project, stemming from a love of both Dungeons and Dragons and basic computer programming. chargen is a work in progress, and some features may be disabled/nonexistent/entirely borked at any given point in time. Issues on GitHub are welcome. Repository is private until the project is semi-functional, at which time I will open it up for community feedback.
